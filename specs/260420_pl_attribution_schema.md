## 1. Purpose

Measure team-level P&L contribution. Each agent logs what it touched. Weekly review surfaces which Pete scan led to which trade, which Edge build enabled which capability. Creates accountability and compounds the team's edge over time.

**Frequency:** Weekly cron — Sunday pre-market (e.g., 6:00 AM CT Sunday)
**Location:** `memory/active/0-14d/YYMMDD_weekly_pl_attribution.md`

---
